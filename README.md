# Diabetes Prediction System using SVM

## Overview

This project is a Machine Learning-based Diabetes Prediction System that predicts whether a patient is at risk of diabetes based on symptoms and medical indicators. The system uses a Support Vector Machine (SVM) classifier and provides a user-friendly graphical interface built with PySide6.

## Features

* Diabetes Risk Prediction
* Support Vector Machine (SVM) Model
* Interactive PySide6 GUI
* Real-Time Prediction
* User-Friendly Interface
* Bilingual Input Labels (English & Bangla)
* Data Preprocessing and Feature Encoding

## Technologies Used

* Python
* Pandas
* Scikit-learn
* PySide6
* NumPy

## Dataset Features

The model uses the following features:

* Age
* Gender
* Polyuria
* Polydipsia
* Sudden Weight Loss
* Weakness
* Polyphagia
* Genital Thrush
* Visual Blurring
* Itching
* Irritability
* Delayed Healing
* Partial Paresis
* Muscle Stiffness
* Alopecia
* Obesity

## Machine Learning Workflow

1. Load Dataset

3. Feature Encoding

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Khadiza3540/diabetes-prediction-system.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python diabetes_prediction.py
```
